# Givi Front

Установка зависимостей

# yarn
yarn install

Development среда
Запускается на http://localhost:5173:

# yarn
yarn dev

Production

# yarn
yarn build

Информация

В проект добавлен плагин svg-symbol-sprite 
для создания svg-спрайта assets/sprites/sprite.svg
из svg-иконок, хранящихся в папке assets/icons.
